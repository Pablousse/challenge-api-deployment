Given input data as a JSON object, this method will return a JSON object with the predicted price.
This method can accept one real estates data or multiple input.
JSON possible input data:

  • area: float, mandatory
  • property-type: string, mandatory
    must be one of the next possible choices
    ["APARTMENT", "HOUSE"]
  • zip-code: int, mandatory,
  • rooms-number: int, optional,
  • building-state: string, optional
    must be one of the next possible choices
    ["AS NEW", "GOOD", "JUST RENOVATED", "TO BE DONE UP",
    "TO RENOVATE", "TO RESTORE"]
  • facade-count: int, optional
Optional argument can be either: null, "Nan", "None" or the argument can be missing like shown in the second example.

Example: 
{
    "data":
        {
            "area": "20",
            "property-type": "APARTMENT",
            "rooms-number": "2",
            "zip-code": "1000",
            "building-state": "TO_RENOVATE",
            "facade-count": 2
        }
}
if you want to input multiple real estate in one request, put them in an array.

Example: 
{
    "data": [
        {
            "area": "20",
            "property-type": "APARTMENT",
            "rooms-number": null,
            "zip-code": "1000",
            "building-state": "TO_RENOVATE",
            "facade-count": "Nan"
        },
        {
            "area": "20",
            "property-type": "APARTMENT",
            "zip-code": "1070",
            "building-state": "TO RENOVATE"
        }
    ]
}
