Introduction

This API takes some real estate input data like area and zipcode and give a prediction of the price of this real estate for the Belgian market

Technical Info

Base URL

The API can be reached by using the following base URL:
https://arnaud-durieux-api-deployment.herokuapp.com/

The Api will return "Alive" if it is up and running

GET predict

what this method does

POST predict

Given input data as a JSON object, this method will return a JSON object with the predicted price.
This method can accept one real estates data or multiple input.
JSON possible input data:

  • area: float, mandatory
  • property-type: string, mandatory
  • zip-code: int, mandatory,
  • rooms-number: int, optionnal,
  • building-state: string, opttionnal
    must be of the next possible choice
    ["AS NEW", "GOOD", "JUST RENOVATED", "TO BE DONE UP",
    "TO RENOVATE", "TO RESTORE"]
  • facade-count: int, optionnal

Example with only one real estate
Python 
import requests
import json

data = {
    "data": {
        "area": "20",
        "property-type": "APARTMENT",
        "rooms-number": "2",
        "zip-code": "1000",
        "building-state": "TO_RENOVATE",
        "facade-count": 2
    }
}

response = requests.post(
    'https://arnaud-durieux-api-deployment.herokuapp.com/predict', json=data)

print("Status code: ", response.status_code)
print("Printing Entire Post Request")
# print(response.json())
print(str(response.content))
